# Header

The module includes `<Header />` component and its subcomponents (`<Header.LeftToolbar />`, `<Header.RightToolbar />`, `<Header.CenterToolbar />`, `<Header.NavMenu />` that serve as placeholders for respective sections of the header).

## `<Header />` Component

The `<Header />` component is a presentational header for the application. It renders the main header container with placeholders for navigation panels to host logo, navigation, and user controls along with a dropdown menu.

**Props:**
- `isNavMenuOpen?: boolean` — Controls whether the navigation menu drawer is expanded.
- `className?: string` — Additional CSS classes.
- `children?: React.ReactNode` — Child components, typically toolbars and nav menu.

**Usage Example:**
```tsx
<Header />
```

**Remarks:**
- Follows design tokens and responsive layout conventions.
- Navigation items and theme controls are customizable via constants.

### `<Header.LeftToolbar />`

Renders the left-aligned toolbar section within the header, typically for navigation or branding.

**Props:**
- `className?: string` — Additional CSS classes.
- `children?: React.ReactNode` — Toolbar contents.

**Usage Example:**
```tsx
<Header>

</Header>
```
